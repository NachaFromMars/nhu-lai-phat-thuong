> 30+ canonical Buddhist text collections ready for citation and research. Full Pali Canon (Theravāda) plus 11 Mahāyāna sutras — 194MB of primary sources across EPUB, PDF, and bilingual JSON formats.

## Overview
nhu-lai-phat-thuong is a multi-tradition Buddhist scripture library for AI research, citation, and composition. Theravāda coverage includes the full Pali Canon: 4 Nikāya in EPUB and PDF (PTS edition), 10 SuttaCentral Sujato EPUBs, Bhikkhu Brahmali Vinaya, and bilara-data JSON with root Pali, English, and Vietnamese translations. Mahāyāna coverage includes 11 key sutras from 84000.co and BuddhaNet spanning Prajnaparamita, Vimalakirti, Ksitigarbha, Medicine Buddha, Lotus, Avatamsaka, and more.

## Features
**Theravāda (Pali Canon):**
- 4 Nikāya EPUB + PDF — PTS unabridged edition
- 10 SuttaCentral EPUBs — Bhikkhu Sujato translation (2025)
- Vinaya — Bhikkhu Brahmali translation
- bilara-data JSON — root Pali + EN + VI translations (segments)

**Mahāyāna (11 sutras):**
- Prajnaparamita, Vimalakirti, Golden Light, Ksitigarbha, Medicine Buddha, Lotus Sutra, Avatamsaka, and more

**Total:** 194MB, 30+ collections

## Usage / Quick Start
Use for: citing Pali Canon suttas, researching doctrine, translating Pali/Sanskrit terms, composing Dharma teachings, comparing Theravāda–Mahāyāna, Buddhist academic writing.
